# Adding documentation for a new endpoint

1. Create a new sheet in `assets/api/FAC API Data Dictionary.xlsx`
1. Create a new CSV file in `src/_data/dictionary_csv/resubmission.csv`
1. Run the script to conver the CSV to JSON, found at `src/_data/README.md`
1. Add the endpoint to the table of contents in `src/api/dictionary.md`
