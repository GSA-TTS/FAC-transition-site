# Resubmitting an audit
In some circumstances, it may be appropriate to resubmit an audit instead of creating a new one in order to make corrections. To see what conditions are necessary and to begin the audit resubmission process, click the 'Resubmit an audit' link, as highlighted below.

<img class="cursor-pointer" src="{{config.baseUrl}}assets/img/walkthrough/walkthrough-18-resubmission.png" width=500 style="margin: 1em; border: 1px solid #555;" aria-controls="image-modal-walkthrough/walkthrough-18-resubmission.png" data-open-modal />
{{ image_modal.modal('walkthrough/walkthrough-18-resubmission.png', 'assets/img/walkthrough/walkthrough-18-resubmission.png', "A screenshot of the FAC 'Audit submissions' page. The 'Resubmit an audit' hyperlink is highlighted.") }}

Carefully read over the criteria listed on this page to determine if your audit is eligible for resubmission. If so, enter the Report ID of the audit you wish to be resubmitted in the input at the bottom of the page and click 'Continue'.

<img class="cursor-pointer" src="{{config.baseUrl}}assets/img/walkthrough/walkthrough-19-resubmission.png" width=500 style="margin: 1em; border: 1px solid #555;" aria-controls="image-modal-walkthrough/walkthrough-19-resubmission.png" data-open-modal />
{{ image_modal.modal('walkthrough/walkthrough-19-resubmission.png', 'assets/img/walkthrough/walkthrough-19-resubmission.png', "A screenshot of the FAC 'Resubmitting an Audit' page. From top to bottom: an overview of resubmission eligibility requirements, a text input for supplying a Report ID, and a 'Continue' button.") }}

If the submission for the Report ID you provided is eligible to be resubmitted, you will be redirected to the 'Submission eligibility' (step #2) page. Because the FAC can pull auditee information from the audit being resubmitted, the 'Auditee information' step is safely skipped. You can now follow the [standard submission steps]({{ config.baseUrl }}audit-resources/submission-guide/starting/) for completing your resubmisison.

<img class="cursor-pointer" src="{{config.baseUrl}}assets/img/walkthrough/walkthrough-20-resubmission.png" width=500 style="margin: 1em; border: 1px solid #555;" aria-controls="image-modal-walkthrough/walkthrough-20-resubmission.png" data-open-modal />
{{ image_modal.modal('walkthrough/walkthrough-20-resubmission.png', 'assets/img/walkthrough/walkthrough-20-resubmission.png', "A screenshot of the FAC submission eligibilty page. One multiple choice question and three true or false questions ask if an entity meets the single audit criteria.") }}

The submission progress page will display a yellow banner to help verify that you are in the process of resubmitting an audit.
